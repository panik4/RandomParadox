## set_variable
- Params: var = <variable> The variable to modify or create. value = <decimal>/<variable> The value to set the variable to. tooltip = localisation_keyLocalisation used by the operation. Optional.
- Example: set_variable = { var = my_variable value = 100 tooltip = set_var_to_100_tt }
- Desc: Sets a variable's value to the specified amount, creating it if not defined.

## set_variable_to_random
- Params: var = <variable> The variable to modify or create. min = <decimal> The minimum possible value, defaults to 0. max = <decimal> The maximum possible value, defaults to 1. integer = <bool> Sets if the variable must be an integer or if it can be decimal. Defaults to false.
- Example: set_variable_to_random = { var = random_num max = 11 integer = yes }
- Desc: Sets a variable's value to the specified amount, creating it if not defined. The result will be greater than or equal than the minimum and strictly less than the maximum.

## clear_variable
- Params: <variable> Variable to clear.
- Example: clear_variable = my_variable
- Desc: Clears the value from the memory entirely.

## modulo_variable
- Params: var = <variable> The variable to modulo. value = <decimal>/<variable> The value to modulo the variable by. tooltip = localisation_keyLocalisation used by the operation. Optional.
- Example: modulo_variable = { var = my_variable value = 50 tooltip = get_modulo_of_var_by_50_tt }

## round_variable
- Params: <variable> The variable to round.
- Example: round_variable = my_variable
- Desc: Rounds the variable towards the closest integer value.

## clamp_variable
- Params: var = <variable> The variable to clamp. min = <decimal>/<variable> The minimum value of the variable after the clamp. max = <decimal>/<variable> The maximum value of the variable after the clamp.
- Example: clamp_variable = { var = my_var min = 0 }
- Desc: Clamps the variable to ensure its value is between the two specified numbers, raising to the minimum if smaller or lowering to the maximum if larger.
