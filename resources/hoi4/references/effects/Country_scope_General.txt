# Country_scope_General
# Source: https://hoi4.paradoxwikis.com/Effect

## set_country_flag
- Params: <flag> An unique string to identify the country flag with. OR flag = <flag> The flag to set. days = <int> Sets the flag to last for the specified amount of days. Optional. value = <int> The new value of the flag on the scale from -2 147 483 648 to 2 147 483 647.
- Example: set_country_flag = my_flag
- Desc: Defines a country flag.
- Since: 1.0

## clr_country_flag
- Params: <flag> The unique string of a country flag to clear.
- Example: clr_country_flag = my_flag
- Desc: Clears a defined country flag.
- Since: 1.0

## modify_country_flag
- Params: flag = <flag> The flag to modify. value = <value> The value to add to the flag. Defaults to 0. days = <int> The amount of days that the flag should last for before being cleared. Optional, defaults to permanent.
- Example: modify_country_flag = { flag = my_flag value = 3 }
- Desc: Adds an integer value to a flag.
- Since: 1.3

## country_event
- Params: id = <event> The event to fire. days = <int> / <variable> Fires the event in the specified number of days. Optional. hours = <int> / <variable> Fires the event in the specified number of hours. Optional. random_hours = <int> / <variable> Adds a random number (between 0 and random_hours, inclusive) of hours to the scheduled fire time. Optional. random_days = <int> / <variable> Adds a random number (between 0 and random_days, inclusive) of days to the scheduled fire time. Optional.
- Example: country_event = { id = my_event.1 days = 10 random_hours = 12 random_days = 10 }
- Desc: Fires the specified event for the current country.
- Since: 1.0

## set_cosmetic_tag
- Params: <string> The cosmetic tag to switch to.
- Example: set_cosmetic_tag = SAF_SOV_communism
- Desc: Makes the current scope use the specified cosmetic tag, changing name and flag.
- Since: 1.3

## drop_cosmetic_tag
- Params: <bool> Boolean.
- Example: drop_cosmetic_tag = yes
- Desc: Makes the current scope drop the current cosmetic tag they are using.
- Since: 1.3

## set_rule
- Params: <rule> Boolean. desc = <localisation key> The localisation used as the description for why the rule is set.
- Example: set_rule = { desc = TAG_my_rule_description can_create_factions = yes }
- Desc: Toggles the special game rules for the current scope. Note: each rule can only be toggled a few times before a reload is required.

## set_party_rule
- Params: ideology = <ideology group> Ideology group of the party. desc = <localisation key> A description used for the rule. Optional, defaults to being the same as default. <rule> = <bool> Rule's new value.
- Example: set_party_rule = { ideology = democratic desc = TAG_my_rule_description can_create_factions = yes }
- Desc: Toggles the special game rules for the current scope's political party.
- Since: 1.12

## add_relation_rule_override
- Params: target = <country> Target of the rule. usage_desc = <localisation key> A description used as the reason for the rule applying. Optional. trigger = <scripted trigger> A scripted trigger deciding when the override should be active. Optional, defaults to always true. <rule> = <bool> Rule's new value.
- Example: add_relation_rule_override = { target = SOV usage_desc = TAG_my_rule_description trigger = my_scripted_trigger can_access_market = yes }
- Desc: Toggles the special game rules for the current scope in diplomacy towards the specified country only, if the trigger is met.
- Since: 1.13

## remove_relation_rule_override
- Params: target = <country> Target of the rule. usage_desc = <localisation key> A description used as the reason for the rule applying. Optional. trigger = <scripted trigger> A scripted trigger for identifying the relation rule. <rule> = <bool> Rule's new value.
- Example: remove_relation_rule_override = { target = SOV usage_desc = TAG_my_rule_description can_access_market = yes }
- Desc: Removes the toggle added with add_relation_rule_override.
- Since: 1.13

## scoped_sound_effect
- Params: <string> A sound reference from an .asset file.
- Example: scoped_sound_effect = "boom"
- Desc: Plays the specified sound once only for the current country.
- Since: 1.6

## scoped_play_song
- Params: <song title from .asset> A music file located in the music folder and .asset
- Example: scoped_play_song = "general_peace_1"
- Desc: Plays an audio track for the specified country only.
- Since: 1.9.3

## goto_province
- Params: <id> The id of the province go to.
- Example: goto_province = 325
- Desc: Moves the camera position over the specified province.
- Since: 1.0

## goto_state
- Params: <state> / <variable> The id of the state go to.
- Example: goto_state = 1
- Desc: Moves the camera position over the specified state.
- Since: 1.0

## change_tag_from
- Params: <country> / <variable> The country to change from.
- Example: change_tag_from = ROOT
- Desc: Switches the player to the current scope from the target scope. Nothing happens if the target scope is controlled by AI.
- Since: 1.0

## reserve_dynamic_country
- Params: <bool>
- Example: reserve_dynamic_country = yes
- Desc: Reserves the dynamic country, making sure that it does not get recycled for civil war even if it does not exist.
- Since: 1.9

## force_update_map_mode
- Params: limit = { ... } Triggers required for the map mode to refresh. Optional. mapmode = <id> The ID of the custom map mode.
- Example: force_update_map_mode = { limit = { is_ai = no } mapmode = my_map_mode }
- Desc: Forcefully refreshes the specified mapmode for the player, rather than waiting for a daily update.
- Since: 1.11

## add_ai_strategy
- Params: type = <type> The type of strategy. id = <country> What country the strategy is against. value = <int> The weighting added by the strategy.
- Example: add_ai_strategy = { type = alliance id = GER value = 200 }
- Desc: Sets an AI strategy for the current scope.
- Since: 1.0

## create_dynamic_country
- Params: original_tag = <tag> The original tag to be used by the country. copy_tag = <tag> If specified, copies stuff from this tag rather than the original tag. <effects> Effects that will be executed on the new dynamic country.
- Example: create_dynamic_country = { original_tag = POL copy_tag = SOV add_political_power = 100 transfer_state = 123 }
- Desc: Creates a new dynamic country, akin to ones used in civil wars.
- Since: 1.9
