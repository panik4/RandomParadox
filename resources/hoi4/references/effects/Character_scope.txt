# Character_scope
# Source: https://hoi4.paradoxwikis.com/Effect

## set_character_flag
- Params: <flag> An unique string to identify the character flag with. OR flag = <flag> The flag to set. days = <int> Sets the flag to last for the specified amount of days. Optional. value = <int> The new value of the flag on the scale from -2 147 483 648 to 2 147 483 647.
- Example: set_character_flag = my_flag
- Desc: Defines a character flag.
- Since: 1.11

## set_character_name
- Params: <localisation key> The name to use.
- Example: set_character_name = GER_my_cool_flag
- Desc: Changes the character's name to the specified localisation key's value.
- Since: 1.11

## modify_character_flag
- Params: flag = <flag> The flag to modify. value = <value> The value to add to the flag. Defaults to 0. days = <int> The amount of days that the flag should last for before being cleared. Optional, defaults to permanent.
- Example: modify_character_flag = { flag = my_flag value = 3 }
- Desc: Adds an integer value to a flag.
- Since: 1.11

## clr_character_flag
- Params: 
- Example: clr_character_flag = <bool>
- Desc: Clears a character flag
- Since: 1.11

## retire
- Params: <bool> Boolean>
- Example: retire = yes
- Desc: Retires the current character (removing them).
- Since: 1.5

## set_nationality
- Params: <country> / <variable> The target country.
- Example: set_nationality = GER
- Desc: Switches the current character to the specified country, giving them the character.
- Since: 1.5

## set_portraits
- Params: character = <character> The character name. Optional if in character scope. Army scope: small = <sprite> The sprite used as an advisor. large = <sprite> The sprite used as a general. Character scope:large = <sprite> The sprite used as a country leader.
- Example: set_portraits = { character = my_character army = { small ="MySmallCharacterGFX" } civilian = { large ="MyLargeCharacterGFX" } }
- Desc: Changes the specified portraits of a character.
- Since: 1.11

## add_trait
- Params: slot = <slot> Slot of the character. Necessary for advisors. ideology = <sub-ideology> Ideology type of the character. Necessary for country leaders. trait = <trait> The trait to add.
- Example: add_trait = { slot = political_advisor trait = really_good_boss }
- Desc: Adds the specified country leader trait to the character.
- Since: 1.11

## remove_trait
- Params: slot = <slot> Slot of the character. Necessary for advisors. ideology = <sub-ideology> Ideology type of the character. Necessary for country leaders. trait = <trait> The trait to remove.
- Example: remove_trait = { slot = political_advisor trait = really_good_boss }
- Desc: Removes the specified trait from the character.
- Since: 1.11

## add_country_leader_role
- Params: character = <character> The character to modify. country_leader = { ... } Country leader role definition promote_leader = <bool> Will promote the leader to be the leader of the assigned party. Optional, defaults to false.
- Example: add_country_leader_role = { character = GER_character_token promote_leader = yes country_leader = { ideology = fascism_type expire = "1965.1.1.1" traits = { war_industrialist } } }
- Desc: Sets the specified character to also act as a country leader, promoting to the party leader if specified.
- Since: 1.11

## add_advisor_role
- Params: advisor = { ... } Advisor role definition activate = <bool> Will activate the advisor (add them directly when the command is run to the countries government). Optional, defaults to false.
- Example: add_advisor_role = { activate = yes advisor = { slot = air_chief cost = 50 idea_token = GER_character_token_air_chief traits = { air_chief_ground_support_2 } } }
- Desc: Sets the specified character to also act as an advisor, activating if specified.
- Since: 1.11

## unit_leader_event
- Params: id = <event> The event to fire. days = <int> / <variable> Fires the event in the specified number of days. Optional. hours = <int> / <variable> Fires the event in the specified number of hours. Optional. random = <int> / <variable> Adds a random number (between 0 and random, inclusive) of hours to the scheduled fire time. Optional. random_days = <int> / <variable> Adds a random number (between 0 and random_days, inclusive) of days to the scheduled fire time. Optional.
- Example: unit_leader_event = { id = my_event.1 days = 10 random = 50 random_days = 10 }
- Desc: Fires the specified event for the owner of the current unit leader.
- Since: 1.5

## operative_leader_event
- Params: id = <event> The event to fire. days = <int> / <variable> Fires the event in the specified number of days. Optional. hours = <int> / <variable> Fires the event in the specified number of hours. Optional. random = <int> / <variable> Adds a random number (between 0 and random, inclusive) of hours to the scheduled fire time. Optional. random_days = <int> / <variable> Adds a random number (between 0 and random_days, inclusive) of days to the scheduled fire time. Optional. originator = <tag> The originator of the event. Optional, defaults to owner of operative. recipient = <tag> The recipient of the event. Optional, defaults to owner of operative. set_from = <tag> Sets the scope of FROM in scripted localization. Optional. set_from_from = <tag> Sets the scope of FROM.FROM in scripted localization. Optional. set_root = <tag> Sets the scope of ROOT in scripted localization. Optional.
- Example: operative_leader_event = { id = my_event.1 originator = POL recipient = GER days = 10 random = 50 random_days = 10 set_from = ENG set_root = SOV set_from_from = FRA }
- Desc: Fires the specified event for the operative.
- Since: 1.9
