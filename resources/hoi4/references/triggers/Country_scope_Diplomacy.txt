# Country_scope_Diplomacy
# Source: https://hoi4.paradoxwikis.com/Triggers

## is_major
- Params: <bool> Boolean.
- Example: is_major = yes
- Desc: Checks if the current scope is considered a Major.
- Since: 1.0

## is_ally_with
- Params: <scope> / <variable> The country to check for.
- Example: is_ally_with = GER
- Desc: Checks if the current scope is an ally (Faction members or subject-master relation).
- Since: 1.0

## is_spymaster
- Params: <bool> Boolean.
- Example: is_spymaster = yes
- Desc: Checks if the current scope is the spymaster of a faction.
- Since: 1.9

## has_non_aggression_pact_with
- Params: <scope> / <variable> The country to check for.
- Example: has_non_aggression_pact_with = GER
- Desc: Checks if the current scope has a non-aggression pact with the specified country.
- Since: 1.0

## is_puppet
- Params: <bool> Boolean.
- Example: is_puppet = yes
- Desc: Returns true if the current country is a puppet.
- Since: 1.0

## is_subject
- Params: <bool> Boolean.
- Example: is_subject = yes
- Desc: Checks if the current scope is a subject.
- Since: 1.0

## has_subject
- Params: <bool> Boolean.
- Example: has_subject = GRE
- Desc: Checks if the country has for subject the given country.
- Since: 1.0

## num_subjects
- Params: <int> The amount to check for.
- Example: num_subjects > 3
- Desc: Checks if the current scope has the specified amount of subjects.
- Since: 1.3

## has_autonomy_state
- Params: <string> The autonomy state to check for.
- Example: has_autonomy_state = autonomy_dominion
- Desc: Checks if the current scope is in the specified autonomous state.
- Since: 1.0

## compare_autonomy_state
- Params: <string> The autonomy state to check for.
- Example: compare_autonomy_state > autonomy_dominion
- Desc: Checks if the current scope's autonomy state min_freedom_level is less or greater than that of the specified autonomy state. The special value "autonomy_free" compares as greater than any autonomy state. If the current scope is not a subject, it is treated as greater than any autonomy state (including "autonomy_free"). With =, checks if the current scope is in the specified autonomous state.
- Since: 1.0

## compare_autonomy_progress_ratio
- Params: <float> The amount to check for.
- Example: compare_autonomy_progress_ratio > 0.5
- Desc: Checks if the current scope autonomy progress is at the specified ratio. If the current scope is not a subject, the ratio is 1.
- Since: 1.3

## has_opinion_modifier
- Params: <string> The opinion modifier to check for.
- Example: has_opinion_modifier = my_modifier
- Desc: Checks if the current scope has the specified opinion modifier.
- Since: 1.0

## has_opinion
- Params: target = <scope> The country to check for. value = <float> The amount to check for.
- Example: has_opinion = { target = GER value > 50 }
- Desc: Checks if the current scope has the specified opinion of the target country.
- Since: 1.0

## has_relation_modifier
- Params: target = <scope> The country to check for. modifier = <modifier> The modifier to check for.
- Example: has_relation_modifier = { target = GER modifier = my_modifier }
- Desc: Checks if the current scope has the specified relation modifier with the specified country.
- Since: 1.0

## has_legitimacy

- Params: sender = <tag> Country which sent forces. value <> <int> Amount of forces.
