# Character_scope_Unit_Leaders
# Source: https://hoi4.paradoxwikis.com/Triggers

## has_unit_leader_flag
- Params: <string> The flag to check for.
- Example: has_unit_leader_flag = my_flag
- Desc: Checks if the current scope has the specified flag.
- Since: 1.5

## has_unit_leader_flag
- Params: flag = <string> The flag to check. value = <int> The flag value to check for. Optional. date = <date> The flag creation date to check for. Optional. days = <int> The duration the flag existed for. Optional.
- Example: has_unit_leader_flag = { flag = my_flag days > 30 date > 1936.6.1 value > 0 }
- Desc: Compares the specified flag's last set date, days since last set, and/or value.
- Since: 1.5

## is_leading_army
