# Combat
# Source: https://hoi4.paradoxwikis.com/Triggers

## hardness
- Params: <float> The amount to check for.
- Example: hardness > 0.5
- Desc: Checks if the current scope has the specified amount of hardness.
- Since: 1.0

- Since: 1.0
