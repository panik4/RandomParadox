# Character_scope_General
# Source: https://hoi4.paradoxwikis.com/Triggers

## is_character
- Params: <scope> Character ID.
- Example: is_character = POL_test_character
- Desc: Checks if the current character's token matches up with the specified one.
- Since: 1.11

## can_be_country_leader
- Params: <bool> Boolean.
- Example: can_be_country_leader = yes
- Desc: Checks if the character in the current scope has a country leader role, active or non-active.
- Since: 1.11

## is_country_leader
- Params: <bool> Boolean.
- Example: is_country_leader = yes
- Desc: Checks if the character in the current scope is the active country leader.
- Since: 1.11

## is_unit_leader
- Params: <bool> Boolean.
- Example: is_unit_leader = yes
- Desc: Checks if the character in the current scope has an active unit leader (Army/Navy leader) role.
- Since: 1.11

## is_advisor

## has_character_flag
- Params: <string> The flag to check for.
- Example: has_character_flag = my_flag
- Desc: Checks if the current scope has the specified flag.
- Since: 1.11

## has_character_flag
- Params: flag = <string> The flag to check. value = <int> The flag value to check for. Optional. date = <date> The flag creation date to check for. Optional. days = <int> The duration the flag existed for. Optional.
- Example: has_character_flag = { flag = my_flag days > 30 date > 1936.6.1 value > 0 }
- Desc: Compares the specified flag's last set date, days since last set, and/or value.
- Since: 1.11

## has_trait
